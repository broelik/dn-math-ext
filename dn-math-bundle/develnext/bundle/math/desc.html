• php\math\BigInteger - класс для работы с большими целыми числами.
  • php\math\BigDecimal - класс для работы с большими дробными числами.
  • php\math\MathContext - вспомогательный класс для работы с BigDecimal.
  • php\math\Math - класс для осуществления математических операция над обычными числами.
